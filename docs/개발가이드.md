# Splearn 개발가이드

## 아키텍처
- 헥사고날 아키텍처
- 도메인 모델 패턴
### 계층
- Domain LAver
- Applicaton Layer
- Apdater Layer

> 외부(Actor) -> 어댑터 -> 어플리케이션 -> 도메인

## 패키지
- domain
- application
  - required : 어플리케이션이 외부를 호출하기 위해 사용
  - provided : 어플리케이션을 외부에 제공하기 위해 사용
- adapter : Actor와 연결하기 위한 인터페이스 정의
  - webapi 
  - persistence
  - integration
  - security
